Native macOS Controls — What Changes

Every custom control replaced with its native SwiftUI equivalent. Click to approve each section.

1

Panel → Liquid Glass NSPanel with Traffic Lights

Delete: Custom xmark close button overlay, TokyoNight.bg.opacity(0.75) background

Replace with:

// FloatingPanelWindow.init styleMask:
styleMask: [.titled, .closable, .miniaturizable,
            .nonactivatingPanel, .fullSizeContentView]
titleVisibility = .hidden
titlebarAppearsTransparent = true
// Traffic lights appear automatically ✓

// ControlsView body:
.glassEffect(.regular, in: .rect(cornerRadius: 14))

Traffic light buttons (red/yellow/green) come free with .titled + .closable + .miniaturizable. The .fullSizeContentView lets content extend behind the title bar. titlebarAppearsTransparent blends it into the glass.

2

Buttons → .glass / .glassProminent

Delete: IconButtonStyle, ThemeButtonStyle, TrayActionStyle — all 4 custom ButtonStyles

Replace with:

// Primary action (Start, Pause):
Button { timerEngine.start() } label: {
    Label("Start", systemImage: "play.fill")
}
.buttonStyle(.glassProminent)
.controlSize(.large)

// Secondary action (Break, Stop):
Button { timerEngine.stop() } label: {
    Label("Stop", systemImage: "stop.fill")
}
.buttonStyle(.glass)

Native glass buttons with SF Symbol icons + text labels. macOS handles all hover, press, and focus states. .controlSize(.large) on primary for visual weight.

3

Timer → SF Pro Ultra Light + System Colors

Delete: .system(size: 49, weight: .light, design: .monospaced), TokyoNight.fg

Replace with:

Text(formatTime(timerEngine.timeRemaining))
    .font(.system(size: 48, weight: .ultraLight))
    .monospacedDigit()
    .foregroundStyle(.primary)   // adapts to dark/light
    .contentTransition(.numericText())

SF Pro (not Mono) with tabular figures via .monospacedDigit(). .contentTransition(.numericText()) gives a native animated digit roll on countdown tick.

4

Duration → Native Stepper

Delete: Custom StepperRow view, custom StepperButtonStyle

Replace with:

Stepper("Focus: \(timerEngine.workDuration)m",
        value: $timerEngine.workDuration,
        in: 5...120, step: 5)
    .font(.system(size: 12))

Stepper("Break: \(timerEngine.breakDuration)m",
        value: $timerEngine.breakDuration,
        in: 1...30, step: 5)
    .font(.system(size: 12))

Native macOS Stepper renders +/- buttons automatically. Keyboard arrow keys work for free. Glass effect inherited from parent container.

5

Sound Picker → Native Menu Picker

Delete: Custom SoundPickerView with manual Menu + HStack layout

Replace with:

Picker(selection: $audioEngine.selectedSound) {
    ForEach(AmbientSound.allCases) { sound in
        Text(sound.displayName).tag(sound)
    }
} label: {
    Label("Sound", systemImage: "speaker.wave.2")
}
.pickerStyle(.menu)

Native dropdown. macOS renders the chevron, selection checkmark, and hover states automatically.

6

Colors → Delete TokyoNight, Use System Semantic

Delete: Entire Theme.swift / TokyoNight enum

Replace with:

.foregroundStyle(.primary)     // main text (timer)
.foregroundStyle(.secondary)   // labels (Focus:, Break:)
.foregroundStyle(.tertiary)    // subtle elements (chevron)

// Dividers:
Divider()   // uses system separator color automatically

Zero custom colors. Automatic dark/light mode. Automatic Increased Contrast accessibility support.
