**Step 1 — Generate placeholder .m4a audio files.** Create `PomodoroApp/Resources/Sounds/` directory. Generate 7 placeholder .m4a files using macOS built-in tools. For each ambient sound, use `say -o /tmp/{name}.aiff "{name} sound placeholder"` then `afconvert /tmp/{name}.aiff PomodoroApp/Resources/Sounds/{name}.m4a -d aac -f m4af`. Generate for: rain, ocean, forest, fireplace, whitenoise, coffeeshop, chime. These are temporary placeholders so the build works -- Rod will replace with real CC0 audio files before shipping.

**Step 2 — Package.swift.** Add `resources: [.process("Resources/")]` to the `PomodoroApp` executableTarget. This is CRITICAL -- without it, `Bundle.main.url(forResource:withExtension:)` returns nil at runtime. Place it after `exclude:` and before `swiftSettings:`.

**Step 3 — AmbientSound enum + AudioEngine.swift.** Create `PomodoroApp/Engine/AudioEngine.swift` with:

`AmbientSound` enum — `String` rawValue, `CaseIterable`, `Identifiable`, `Sendable`:
- Cases: `.rain`, `.ocean`, `.forest`, `.fireplace`, `.whitenoise`, `.coffeeshop`
- `var displayName: String` — human-readable names ("White Noise", "Coffee Shop", etc.)
- `var filename: String { rawValue }` — matches filenames without extension
- `var id: String { rawValue }`

`AudioEngine` class — `@MainActor final class AudioEngine: ObservableObject`:
- `@Published var selectedSound: AmbientSound` — initialized from `UserDefaults.standard.string(forKey: "selectedSound")` falling back to `.rain`
- `private var ambientPlayer: AVAudioPlayer?` — STORED property (not local variable -- ARC will deallocate a local and audio stops after 1-2 seconds)
- `private var chimePlayer: AVAudioPlayer?` — STORED property, same reason
- `func startAmbient()` — loads selected sound from `Bundle.main.url(forResource:withExtension: "m4a")`, sets `numberOfLoops = -1` (infinite loop), `volume = 0.7`, calls `prepareToPlay()` then `play()`. Guard on nil URL with `assertionFailure`. Stop existing player first (`ambientPlayer?.stop(); ambientPlayer = nil`).
- `func stopAmbient()` — `ambientPlayer?.stop(); ambientPlayer = nil`
- `func playChime()` — loads "chime.m4a" from bundle, `numberOfLoops = 0` (play once), `prepareToPlay()` then `play()`. Stop existing chime first.
- `func select(_ sound: AmbientSound)` — sets `selectedSound`, persists to `UserDefaults.standard.set(sound.rawValue, forKey: "selectedSound")`. If `ambientPlayer?.isPlaying == true`, calls `startAmbient()` to switch track immediately.

Import `AVFoundation` and `SwiftUI` (for ObservableObject + Published).

Do NOT import or use `AVAudioSession` -- it does not exist on macOS.
